# Phase 3: Control Flow Graph Analysis

## Overview
This module implements Control Flow Graph (CFG) based similarity analysis for plagiarism detection in source code.

## Installation

pip install -r requirements.txt

## How to run

python -m phase3.main -f1 phase1/tests/test_python/code1.py -f2 phase1/tests/test_python/code2.py  --html --html-compare

After running the above command, results are saved in phase3/results
